
# 📝 Notes App (MERN Stack)

A full-stack Notes Application built using the MERN stack (MongoDB, Express, React, Node.js) with a modern UI and full CRUD functionality.

---

## 🚀 Features

- ➕ Add Notes  
- 📄 View Notes  
- ✏️ Edit Notes  
- ❌ Delete Notes  
- 🎨 Modern UI (Glassmorphism + Gradient + Hover Effects)

---

## 🛠️ Tech Stack

- **Frontend:** React  
- **Backend:** Node.js, Express  
- **Database:** MongoDB Atlas  
- **Styling:** CSS (Glassmorphism + Responsive Design)

---

## 📂 Project Structure
simple-notes-app/
├── backend/
├── frontend/

---

## ⚙️ Setup Instructions

### 1️⃣ Clone the repository
git clone https://github.com/qudsia-ux/simple-notes-app.git

cd simple-notes-app

---

### 2️⃣ Backend Setup
cd backend
npm install
npm run dev

---

### 3️⃣ Frontend Setup
cd frontend
npm frontend
npm start

---

## 🔐 Environment Variables

Create a `.env` file inside `backend/`:
MONGO_URI=mongodb_connection_string
PORT=5000
